# WhatDog
An iOS Dog type recognition app built with CoreML and MobileNet Model(https://developer.apple.com/machine-learning/)

## Requirements
- iOS 11.0+
- Xcode 9

## Installation

- run `pod install` under source repo
- Open WhatDog.xcworkspace
- Build App
